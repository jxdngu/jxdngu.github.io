Hexo搭建

环境准备

安装nodejs

  1. 在官网下载二进制文件 https://nodejs.org/en/download

  2. 将二进制文件 node-vxx-linux-x64.tar.xz 解压到想要安装的路径，我使用的是/usr/local/lib/nodejs

  3. 将nodejs加入环境变量
  4. 验证nodejs是否安装成功

安装hexo

npm install hexo-cli -g

搭建博客

初始化

  1. 创建新的文件夹mkdir Bolg && cd Blog
  2. 初始化hexo init

部署到github

  1. 在github创建一个与用户名相同的仓库, 例如egbertngu.github.io
  2. 安装github部署插件 npm install --save hexo-deployer-git(每建一个仓库都需要安装)
  3. 修改博客根目录下的配置_config.yml
  4. 执行 hexo clean && hexo deploy

  5. 常用指令

    启动本地博客在线预览: hexo s 或 hexo server
    新建博客 hexo new xxx, 创建的文件在source/_posts目录中, 注文件名不要加.md
    生成博客 hexo generate
